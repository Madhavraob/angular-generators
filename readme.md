
--Install prerequisites
  Node.js 6 or higher
  npm 3 or higher (which comes bundled with Node)

Steps for generating (scafolding) a project using Yeoman generator

1.Instal yeoman generator globally
1.1 npm install --global yo

2.git clone https://github.com/Madhavraob/angular-generators.git

3.cd ./angular-generators/angular-gen

4.Run below command in above folder (./angular-generators/angular-gen)
4.1 npm link

5.move to a new folder where angualr project should be scafolded and run below command
5.1 yo angular-gen sample
